**Процесс получения цифрового изображения**

Цифровое изображение представляет собой двумерную матрицу, состоящую из пикселей, каждый из которых имеет определенное значение яркости. Процесс получения цифрового изображения можно разделить на следующие этапы:

* **Модель диерковой камеры**

Модель диерковой камеры является упрощенной моделью процесса получения изображения. В этой модели предполагается, что изображение формируется путем пропускания света через отверстие в непрозрачном экране. Положение отверстия определяет точку, в которой будет получен пиксель изображения.

* **Трансформации из 3D в 2D**

В реальных условиях изображение формируется из трехмерного пространства. Для получения цифрового изображения необходимо преобразовать трехмерное изображение в двумерное. Это можно сделать с помощью следующих преобразований:

    * **Проекция**

Проекция - это преобразование, которое преобразует точку в трехмерном пространстве в точку в двухмерном пространстве. Существует множество различных типов проекций, которые можно использовать для получения цифрового изображения.

    * **Ограничение поля зрения**

Ограничение поля зрения означает, что только часть трехмерного пространства будет отображаться на цифровом изображении. Это можно сделать с помощью объектива камеры.

* **Что такое изображение функции**

Изображение функции - это отображение, которое отображает точки трехмерного пространства в значения яркости.

* **Разница между непрерывной и дискретной (цифровой) изображением функции**

Непрерывная изображение функции - это функция, которая может принимать любые значения яркости. Дискретная (цифровая) изображение функция - это функция, которая может принимать только ограниченное количество значений яркости.

* **Векторизация, квантовизация**

Векторизация - это процесс дискретизации изображения функции. Это означает, что каждому пикселю изображения присваивается значение яркости.

Квантизация - это процесс округления значений яркости до ограниченного количества значений.

* **Алиасинг**

Алиасинг - это артефакт, который возникает в результате векторизации или квантования изображения функции. Алиасинг может привести к искажению изображения.

* **Связь с теоремой Шеннона**

Теорема Шеннона утверждает, что для того, чтобы избежать алиасинга, необходимо, чтобы частота дискретизации изображения была не менее вдвое больше частоты пространственных частот изображения.

* **Получение цветного изображения и демозаикование**

Цветное изображение формируется из трех изображений, полученных с помощью трех различных светофильтров: красного, зеленого и синего. Этот процесс называется демозаикированием.

* **Перевод цветного изображения в черно-белое**

Перевод цветного изображения в черно-белое может быть выполнен с помощью следующих методов:

    * **Коэффициент яркости**

Этот метод основан на использовании коэффициента яркости для каждого из трех каналов изображения.

    * **Смешение цветов**

Этот метод основан на смешивании цветов для получения черно-белого изображения.

**Дополнительная информация**

В процессе получения цифрового изображения необходимо учитывать следующие факторы:

* **Разрешение**

Разрешение - это количество пикселей в изображении. Чем выше разрешение, тем больше деталей будет видно на изображении.

* **Глубина цвета**

Глубина цвета - это количество бит, используемых для представления значения яркости каждого пикселя. Чем выше глубина цвета, тем точнее будет представлен диапазон яркостей.

* **Формат изображения**

Формат изображения определяет способ хранения данных изображения. Существует множество различных форматов изображений, каждый из которых имеет свои преимущества и недостатки.

**Заключение**

Процесс получения цифрового изображения является сложным и многоэтапным. Для получения качественного изображения необходимо учитывать множество факторов.

### 2.	Fundamentálna matica kamery, jej rozklad na vonkajšiu a vnútornú maticu, čo popisujú jednotlivé matice, princíp procesu kalibrácie kamery, čo je reprojekčná chyba, skreslenie - radiálne a tangeciálne. 

**Основная матрица камеры**

Основная матрица камеры - это матрица, которая описывает проективное преобразование между двумя изображениями, полученными с помощью двух камер. Матрица имеет размер 3x3 и состоит из следующих элементов:

* **A11, A12, A13** - элементы, определяющие положение камеры в пространстве.
* **A21, A22, A23** - элементы, определяющие направление взгляда камеры.
* **A31, A32, A33** - элементы, определяющие масштаб изображения.

**Разложение на внешнюю и внутреннюю матрицы**

Основную матрицу камеры можно разложить на две матрицы: внешнюю и внутреннюю. Внешняя матрица описывает положение камер в пространстве, а внутренняя матрица описывает свойства объектива камеры.

**Внешняя матрица**

Внешняя матрица имеет размер 3x3 и определяется следующим образом:

```
R = [R11 R12 R13]
  [R21 R22 R23]
  [R31 R32 R33]
```

где R11, R12, R13, R21, R22, R23, R31, R32, R33 - элементы матрицы поворота, а t1, t2, t3 - элементы вектора смещения.

**Внутренняя матрица**

Внутренняя матрица имеет размер 3x3 и определяется следующим образом:

```
K = [fx 0 cx]
  [0 fy cy]
  [0 0 1]
```

где fx и fy - фокусные расстояния камеры по горизонтали и вертикали, cx и cy - координаты оптического центра камеры.

**Что описывают отдельные матрицы**

* **Внешняя матрица**

Внешняя матрица описывает положение камер в пространстве. Элементы R11, R12, R13 определяют поворот камеры вокруг оси x, элементы R21, R22, R23 определяют поворот камеры вокруг оси y, а элементы R31, R32, R33 определяют поворот камеры вокруг оси z. Элементы t1, t2, t3 определяют положение камеры в пространстве.

* **Внутренняя матрица**

Внутренняя матрица описывает свойства объектива камеры. Элементы fx и fy определяют фокусные расстояния камеры по горизонтали и вертикали, а элементы cx и cy определяют координаты оптического центра камеры.

**Принцип процесса калибрации камеры**

Процесс калибрации камеры - это процесс определения элементов внешней и внутренней матриц камеры. Для этого необходимо получить несколько пар изображений, снятых с двух камер. Изображения должны быть сняты с известными положениями камер в пространстве.

**Репроекционная ошибка**

Репроекционная ошибка - это ошибка, которая возникает в результате применения основной матрицы камеры для преобразования координат объекта в координаты изображения. Репроекционная ошибка может быть вызвана различными факторами, включая неточность калибровки камеры, искажение объектива и шум изображения.

**Смещение - радиальное и тангенциальное**

Смещение - это искажение изображения, которое возникает в результате неидеальности объектива камеры. Смещение может быть радиальным или тангенциальным.

* **Радиальное смещение**

Радиальное смещение возникает из-за того, что фокусное расстояние объектива не является постоянным по всей площади изображения. Это приводит к тому, что объекты, расположенные близко к центру изображения, отображаются более точно, чем объекты, расположенные близко к краям изображения.

* **Тангенциальное смещение**

Тангенциальное смещение возникает из-за того, что оптический центр объектива не является центром изображения. Это приводит к тому, что объекты, расположенные близко к центру изображения, отображаются с искажением, которое увеличивается по мере удаления объекта от центра изображения.

**Заключение**

Основная матрица камеры - это важный инструмент в компьютерном зрении. Она используется для решения различных задач, включая сопоставление изображений, определение глубины и восстановление 3D-модели.
